# Claude Deep Suite

Unified marketplace for Deep plugins.

## Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [deep-work](https://github.com/Sungmin-Cho/claude-deep-work) | Evidence-Driven Development Protocol | v5.6.0 |
| [deep-wiki](https://github.com/Sungmin-Cho/claude-deep-wiki) | LLM-managed markdown wiki | v1.0.1 |
| [deep-evolve](https://github.com/Sungmin-Cho/claude-deep-evolve) | Autonomous Experimentation Protocol | v1.0.0 |

## Install

```bash
claude plugin marketplace add github:Sungmin-Cho/claude-deep-suite
```

Then install individual plugins:

```bash
claude plugin install deep-work@claude-deep-suite
claude plugin install deep-wiki@claude-deep-suite
claude plugin install deep-evolve@claude-deep-suite
```
